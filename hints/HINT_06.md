# 힌트 06: 자가 진단 — 이제 뭘 개선해야 하는가

`POST /api/chat`이 동작합니다. 정확도도 측정했습니다.

지금 만든 것의 상태를 솔직하게 평가해보세요.

## 증상

- 동작은 하는데 "이게 맞나?" 싶다
- 정확도가 기대보다 낮다
- 어디를 고쳐야 할지 감이 안 잡힌다

## 자가 진단 체크리스트

아래 항목을 직접 확인해보세요:

```
[ ] 정확도: 테스트 질문 150개 중 몇 %가 정확한가?
[ ] 비용: 150개 요청에 얼마가 드는가? 1,000개라면?
[ ] 속도: 평균 응답 시간은?
[ ] 실패 패턴: 어떤 유형의 질문에서 주로 틀리는가?
[ ] 데이터 활용: Layer 1/2/3를 모두 쓰고 있는가?
```

## 일반적인 결과

대부분의 구현이 이 범위에 있습니다:

| 지표 | 일반적인 결과 |
|------|-------------|
| 전체 정확도 | 25-40% |
| Easy 질문 | 60-70% |
| Medium 질문 | 15-25% |
| Hard 질문 | 5-15% |

이 수치는 **실패가 아닙니다.** 이것이 출발점입니다.
