# 힌트 05: 품질 측정 — 답변이 맞는지 어떻게 아는가

API가 동작합니다. 질문을 보내면 답변이 옵니다.
그런데 이 답변이 맞는지 어떻게 확인하나요?

## 증상

- 답변이 나오긴 하는데, 맞는지 틀리는지 모르겠다
- 하나씩 눈으로 확인하는 건 한계가 있다
- 코드를 바꿀 때마다 전체를 다시 확인해야 한다

## 왜 이게 어려운가

```json
{
  "question": "마켓플레이스 상품도 14일 반품 가능한가요?",
  "answer": "네, 모든 상품은 14일 이내에 반품하실 수 있습니다."
}
```

이 답변은 **틀렸습니다.** 마켓플레이스 상품은 셀러별 정책을 따릅니다.
그런데 API 응답만 보면 그럴듯합니다.

테스트 질문이 150개인데, 수동 검토는:
- 시간이 너무 오래 걸리고
- 검토자마다 기준이 다르고
- 변경할 때마다 다시 해야 합니다
